计算接口一：

地址：/cal/compute/

请求方法：POST，参数格式：json

            {
                "Mg": 0.25,      # 元素含量
                "Cu": 0.36,
                .....
                "prop": "密度"    # 要计算的属性
            }

响应结果格式

            {
                code: 1                     # 状态码，1-成功， 0失白， -1-请求驳回
                data: 26.590038553949288    # 计算结果
                msg: "Success"              # 提示消息
            }
